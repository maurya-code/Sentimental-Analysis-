About this project

SentiAna was built using Python + Django with the VADER (Valence Aware Dictionary and Sentiment Reasoner) library for backend sentiment scoring. This portfolio version runs entirely in the browser using a JavaScript port of the same VADER algorithm.

The system handles negation ("not good" → negative), booster words ("very amazing" → higher score), and returns a compound score from -1 to +1.

Python Django VADER NLP SQLite JavaScript HTML/CSS

Score Thresholds

≥ +0.05 Positive
-0.05 to +0.05 Neutral
≤ -0.05 Negative
